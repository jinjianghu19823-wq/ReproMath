# Notebook Scaffold

Run:

```bash
repromath scaffold notebook --topic "truncated HOSVD"
```

The default output is:

```text
notebooks/truncated-hosvd.ipynb
```

The notebook contains study-first sections:

- Title
- Motivation
- Mathematical setup
- Definitions and notation
- Algorithm
- Naive implementation
- Stable or recommended implementation
- Numerical experiment
- Diagnostics
- Interpretation
- Try it yourself
- References

It also includes starter NumPy/Matplotlib code, a fixed random seed, toy data,
an algorithm placeholder, and diagnostics placeholders.
