# ReproMath Documentation

ReproMath is a lightweight CLI toolkit for reproducible mathematical research
projects. It helps keep notebooks, figures, LaTeX files, QA reports, and
source-to-output mappings close together.

Start with the [Quickstart](quickstart.md), then read the focused pages for
LaTeX QA, notebook scaffolding, figure recipes, and project metadata. To see
how the pieces fit together, read the
[Low-Rank Tensor PDE showcase](showcase-low-rank-tensor-pde.md).

Before tagging a release, use the [Release Checklist](release-checklist.md).
